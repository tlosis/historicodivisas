---pasos para la instalacion---

1)descargar todo el codigo en la carpeta raiz del servidor
2)crear la base de datos del sistema. usar el archivo "historicodivisas.sql"
3)Configurar los datos de conexion de la base de datos en el archivo "config.php"
4)Dirigirse en su mnavegador web a la ruta raiz donde coloco el proyecto. Buscar el archivo index.php
